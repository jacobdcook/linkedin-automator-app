# LinkedIn Automator (releases)

Binary releases for **[LinkedIn Automator](https://github.com/jacobdcook/linkedin-automator)** — a desktop hub for LinkedIn job-search workflows: people search scraping, connection batching with approval, feed engagement, and job scraping.

**This repository does not contain source code.** It holds the public **license**, **documentation**, and **GitHub Releases** zips only.

## Features (summary)

- People search batch scraper → Excel-oriented workflow  
- Daily connection batch generation with AI or template notes (you approve sends)  
- Follow-up messenger, acceptor, feed engager  
- Job scraper tracks + optional enrich / study guide tooling  
- **Simple Mode** in the Hub: four large actions for non-technical users
